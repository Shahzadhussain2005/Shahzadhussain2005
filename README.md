### 🚀 Featured Projects

#### 🔹 [Lean RAG Chatbot](https://github.com/Shahzadhussain2005/lean-rag)
A cost-optimized RAG chatbot built with a **4-layer caching architecture**, query compression, token trimming, and sentence-aware chunking — designed to cut LLM API costs significantly without sacrificing retrieval accuracy. Built for real-world deployment economics, not just proof-of-concept.

#### 🔹 [Shopify RAG Chatbot](https://github.com/Shahzadhussain2005/Shopify-Rag)
An end-to-end RAG system that **crawls and ingests live Shopify store data**, then answers natural-language business questions on top of it — including a **SQL-to-text layer** that lets users ask things like *"what was our best-selling item last month?"* and get accurate, data-grounded answers.

#### 🔹 Legal Document RAG System
A document-intelligence RAG pipeline built to organize and retrieve information from **legal documents, including scanned files processed via OCR**. Documents are chunked, embedded (Pinecone / FAISS / ChromaDB), and tagged with metadata — including document type, date, and confidentiality tier — to support accurate, filtered retrieval for legal drafting and reference workflows.

#### 🔹 Medibot — AI Medical Assistant Chatbot
An AI-powered assistant designed to answer medical/health-related queries by grounding responses in curated medical knowledge sources, prioritizing accuracy and safe, non-hallucinated responses over generic LLM output.

#### 🔹 Karachi AQI — End-to-End ML Pipeline
A full machine learning pipeline for **air quality index prediction in Karachi** — covering data collection, preprocessing, feature engineering, model training, and evaluation, built as a complete real-world ML system rather than a notebook-only experiment.
